# Data Sources Evaluation

本文檔記錄各數據源的評估結果，用於選擇最佳的 3 年歷史數據獲取方案。

## 目標需求

- **時間範圍**: 最近 3 年 (2022-2025)
- **數據類型**: 股價、新聞、財報
- **股票範圍**: ~30-70 支核心股票 (見 `config/tickers_core.json`)

#### EOD 數據說明

**EOD (End of Day) = 完整的日線 OHLCV 數據**，不只是收盤價：

```json
{
  "date": "2025-12-05",
  "open": 280.54,      // 開盤價
  "high": 281.14,      // 當日最高
  "low": 278.05,       // 當日最低
  "close": 278.78,     // 收盤價
  "volume": 47265845,  // 成交量
  "adj_close": 278.78  // 調整後收盤價 (股息/拆股)
}
```

**EOD vs 分鐘級數據選擇指南：**

| 策略類型 | EOD OHLCV | 分鐘級 | 說明 |
|---------|-----------|-------|------|
| 日線/波段策略 | ✅ 足夠 | 不需要 | MA, RSI, MACD 等指標 |
| 日內交易 | ❌ 不夠 | ✅ 必須 | 需要盤中價格變化 |
| 新聞事件分析 | ⚠️ 有限 | ✅ 推薦 | 財報發布後即時反應 |
| 高頻交易 | ❌ | ❌ | 需要 Tick 數據 |

### 新聞數據實測比較 (AAPL, 2025-12-12)

| 平台 | 7天新聞量 | 30天新聞量 | 內容完整度 | 情緒分析品質 | 目標股票精確度 |
|-----|----------|-----------|-----------|-------------|--------------|
| **Finnhub** | **190篇** | 250篇 | 標題+摘要+URL | ❌ 無 | ⭐⭐⭐ 精確 (`related`只有AAPL) |
| **Alpha Vantage** | 50篇 | - | 標題+摘要+URL | ✅ **最詳細** | ⭐ 混雜 (需用`relevance_score`過濾) |
| **Polygon** | 45篇 | - | 標題+URL | ✅ 簡單(-1/0/1) | ⭐⭐ 中等 |
| **yfinance** | 10篇 | - | ❌ 標題為null | ❌ 無 | - |

#### Finnhub 新聞特點 (推薦)
- 數量最多：7天 190 篇
- 每篇有完整 `description` (約 200 字摘要)
- `related` 欄位通常只有目標股票，精確度高
- 來源多樣：Yahoo, SeekingAlpha, Reuters, Benzinga

#### Alpha Vantage 新聞特點 (情緒分析最佳)
- AI 情緒分析最詳細：
  ```json
  {
    "overall_sentiment_score": 0.312,
    "overall_sentiment_label": "Somewhat-Bullish",
    "ticker_sentiments": [
      {"ticker": "AAPL", "relevance_score": "0.705", "sentiment_score": "0.306"}
    ]
  }
  ```
- **缺點**：搜尋 AAPL 會回傳提及 AAPL 的所有新聞，包含大量其他股票為主的文章
- **建議**：用 `relevance_score > 0.7` 過濾

#### Polygon 新聞特點
- 情緒標籤簡單：-1 (負面), 0 (中性), 1 (正面)
- 有 `sentiment_reasoning` 欄位 (AI 解釋)
- 來源以 Motley Fool, Investing.com 為主
