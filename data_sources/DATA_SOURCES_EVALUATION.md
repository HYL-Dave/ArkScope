# Data Sources Evaluation

本文檔記錄各數據源的評估結果，用於選擇最佳的 3 年歷史數據獲取方案。

## 目標需求

- **時間範圍**: 最近 3 年 (2022-2025)
- **數據類型**: 股價、新聞、財報
- **股票範圍**: ~30-70 支核心股票 (見 `config/tickers_core.json`)

### 4. Polygon.io (股價分鐘級 + 新聞情緒) (已更名 Massive.com)

| 項目 | 免費方案 | Starter $99/月 | Developer $199/月 |
|------|---------|----------------|-------------------|
| API 限制 | 5 calls/min | 無限制 | 無限制 |
| 股價歷史 | **2 年 (分鐘級)** | 10 年 | 15 年 |
| EOD 數據 | ✅ | ✅ | ✅ |
| 分鐘/秒級 | ✅ (2年) | ✅ | ✅ |
| 即時數據 | 15 分鐘延遲 | 15 分鐘延遲 | **即時** |
| 新聞歷史 | ✅ (含情緒) | ✅ | ✅ |
| Tick 數據 | ❌ | ✅ (2004起) | ✅ |

**測試結果 (2025-12-12):**
- ✅ API 連接正常
- ✅ 公司資料 (市值、上市日期等)
- ✅ **2 年分鐘級數據** - 實測 660 根/天
- ✅ **新聞 API 含 AI 情緒分析** - 7天內 45 篇
- ⚠️ 最近日線數據有延遲

**特點**:
- 免費方案意外地包含 2 年分鐘級歷史數據
- 新聞自帶情緒標籤 (positive/neutral/negative)
- 付費方案無每日上限，只有每分鐘限制

Sources: [Polygon Pricing](https://polygon.io/pricing), [Polygon Blog](https://polygon.io/blog/flat-files)

## 推薦方案

### 方案 A: 純免費 - 日線策略 (成本: $0)

```
股價 EOD:    Tiingo (30+ 年) ✅

             ⚠️ 缺口: 2021-2023
```

**適用**: 日線/週線策略、研究、原型開發

### 方案 A+: 純免費 - 日內策略 (成本: $0) ⭐ 推薦

## 免費 vs 付費差異分析

### 股價數據差異

| 特性 | 免費 (Tiingo) | 付費 (Polygon) |
|------|--------------|----------------|
| 數據頻率 | **日線 (EOD)** | **分鐘/秒級** |
| 延遲 | T+1 (收盤後) | 即時 |
| 歷史深度 | 30+ 年 | 完整 |
| 盤前盤後 | ❌ | ✅ |
| 調整價格 | ✅ | ✅ |

**影響**:
- 日內策略需要分鐘級數據 → 需付費
- 日線/週線策略 → 免費足夠

### 新聞數據差異

| 特性 | 免費 (Finnhub) | 付費 (Polygon) |
|------|---------------|----------------|
| 歷史深度 | **1 年** | **5+ 年** |
| 來源數量 | 中等 | 多 |
| 更新延遲 | 分鐘級 | 秒級 |
| 全文內容 | 摘要為主 | 完整 |

**影響**:
- 訓練長期模型需要多年歷史 → 考慮付費或用 FNSPID 補齊
- 即時交易訊號 → 免費足夠
