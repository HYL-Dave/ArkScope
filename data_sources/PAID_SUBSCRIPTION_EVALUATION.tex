\section{Financial Datasets}

\subsection{獨特優勢: Segmented Revenue API}

\begin{tcolorbox}[colback=fdsblue!5,colframe=fdsblue,title={其他 API 少見的功能}]
直接從 XBRL 格式 10-K/10-Q 解析:
\begin{itemize}[leftmargin=*]
    \item \textbf{產品線營收細分} (iPhone, Mac, iPad...)
    \item \textbf{業務部門營收} (不同事業群)
    \item \textbf{地理區域營收} (Americas, Europe, Greater China...)
    \item 保留原始 XBRL 標籤，可追溯性高
\end{itemize}
\end{tcolorbox}

\textbf{應用場景}:
\begin{itemize}
    \item 分析 Apple 的 iPhone vs Mac 營收趨勢
    \item 追蹤公司區域曝險 (中國營收佔比)
    \item 識別成長/衰退的業務線
    \item \textbf{PEAD 策略}: 哪些產品線驅動 Earnings Surprise
\end{itemize}
