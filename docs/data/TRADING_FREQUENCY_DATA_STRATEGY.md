## 交易頻率分類框架

### 七級分類系統 (含 L3 細分)

| 級別 | 名稱 | 持倉時間 | 決策頻率 | 數據延遲容忍 | AI Agent 適用 |
|:----:|------|---------|---------|:------------:|:-------------:|
| **L1** | 高頻交易 (HFT) | 毫秒-秒 | 每秒數次 | 微秒級 | ❌ 不適用 |
| **L2** | 超短線 (Scalping) | 秒-分鐘 | 每分鐘 | 秒級 | ❌ 不適用 |
| **L3a** | 快速日內 | 1-5 分鐘 | 每分鐘數次 | < 1 分鐘 | ⚠️ 有限適用 |
| **L3b** | 標準日內 ⭐ | 15-30 分鐘 | 每 15-30 分鐘 | **15 分鐘可** | ✅ **甜蜜點** |
| **L4** | 短波段 (Short Swing) | 小時-數天 | 每數小時-每天 | 分鐘-小時級 | ✅ 適用 |
| **L5** | 中波段 (Swing) | 數天-數週 | 每天-每週 | 小時-天級 | ✅ 最適用 |
| **L6** | 長線持有 (Position) | 數週-數月 | 每週-每月 | 天級 | ✅ 適用 |

> ⭐ **重要發現**: L3b (15 分鐘區間) 是 AI Agent 的甜蜜點！
> - 保留日內交易機會 (每日 26 個決策點)
> - 成本與 L4-L5 相同 (不需即時數據)
> - LLM 有充裕分析時間 (每循環 5-10 分鐘)

## 各頻率層級詳細分析

### L1: 高頻交易 (HFT)

```
持倉時間: 毫秒 - 秒
決策頻率: 每秒數十至數千次
典型策略: 做市、套利、統計套利

數據需求:
├── 延遲: < 1 毫秒
├── 類型: Level 2/3 訂單簿、Tick 數據
├── 成本: $10,000+/月 (co-location + 專線)
└── 基礎設施: FPGA、專用伺服器、交易所共置

AI Agent 適用性: ❌ 完全不適用
├── 原因: LLM 推理延遲 (~100ms-1s) 遠超過交易窗口
├── 競爭對手: 專業量化基金、做市商
└── 進入門檻: 極高 (技術 + 資本)
```

### L2: 超短線 (Scalping)

```
持倉時間: 秒 - 分鐘
決策頻率: 每分鐘數次
典型策略: 動量捕捉、價差交易、新聞反應

數據需求:
├── 延遲: < 1 秒
├── 類型: 即時報價、Options Flow、Level 2
├── 成本: $200-500/月
└── 基礎設施: 低延遲 API、專用網路

AI Agent 適用性: ❌ 不適用
├── 原因: 決策窗口太短，LLM 無法提供有意義分析
├── 競爭對手: 專業日內交易者、算法
└── 關鍵成功因素: 速度 > 分析深度
```

### L3: 日內交易 (Day Trading)

```
持倉時間: 分鐘 - 小時 (當日平倉)
決策頻率: 每小時數次
典型策略: 趨勢跟隨、突破交易、Options Scalping

數據需求:
├── 延遲: < 1 分鐘
├── 類型: 即時報價、Options Flow、Dark Pool、即時新聞
├── 成本: $150-300/月
│   ├── Polygon 即時: $29
│   ├── Unusual Whales: $48
│   ├── FlowAlgo/替代: $100
│   └── Benzinga 即時新聞: ~$99 (可選)
└── 基礎設施: 穩定網路、即時數據 feed

AI Agent 適用性: ⚠️ 有限適用
├── 可用於: 開盤前分析、盤中警報篩選
├── 不適用於: 即時交易決策
├── LLM 角色: 輔助而非主導
└── 建議: 結合規則引擎 + LLM 預分析
```

### L4: 短波段 (Short Swing) ⭐ 你的目標區間

```
持倉時間: 數小時 - 數天
決策頻率: 每數小時 - 每天
典型策略: 事件驅動、盈餘反應、技術突破確認

數據需求:
├── 延遲: 分鐘 - 小時 (可接受 15 分鐘延遲)
├── 類型:
│   ├── 必要: 延遲報價、新聞、內線交易、財報日曆
│   ├── 高價值: Options Flow (可批次抓取)、Short Interest
│   └── 可選: 即時報價 (用於執行)
├── 成本: $50-150/月
│   ├── 免費: Finnhub 新聞、SEC 內線、Tiingo 價格
│   ├── Polygon $29: 延遲 Options + 新聞情緒
│   └── Unusual Whales $48: Options Flow (可每小時抓取)
└── 基礎設施: 標準雲端 + 排程任務

AI Agent 適用性: ✅ 適用
├── 決策窗口: 數小時，足夠 LLM 深度分析
├── 預抓比例: ~60-70%
├── LLM 角色: 主導分析，輔助決策
├── 典型工作流:
│   ├── 每小時: 抓取 Options Flow、新聞
│   ├── LLM 分析: 識別機會、評估風險
│   └── 決策: 數小時內執行
└── Alpha 來源: 新聞反應 (4hr-5天)、Options Flow 跟隨
```

### L5: 中波段 (Swing) ⭐ 最適合 AI Agent

```
持倉時間: 數天 - 數週
決策頻率: 每天 - 每週
典型策略: PEAD、內線跟隨、國會交易跟隨、分析師升降級

數據需求:
├── 延遲: 小時 - 天 (EOD 數據足夠)
├── 類型:
│   ├── 核心: EOD 價格、財報、內線交易、國會交易
│   ├── 高價值: 分析師預估變動、Short Interest
│   └── 增強: 新聞情緒歷史、Options 異常歷史
├── 成本: $0-80/月
│   ├── 免費: SEC、Finnhub、Quiver、Tiingo、FINRA
│   ├── Financial Datasets $30: Segmented Revenue
│   └── 分析師預估 ~$50: 可選
└── 基礎設施: 任意雲端/本地，每日批次

AI Agent 適用性: ✅ 最適用
├── 決策窗口: 數天，充分發揮 LLM 深度分析優勢
├── 預抓比例: ~90%+ (幾乎所有數據可預抓)
├── LLM 角色: 完全主導
├── 典型工作流:
│   ├── 每日凌晨: 批次抓取所有數據
│   ├── LLM 分析: 深度財報分析、模式識別
│   └── 決策: 開盤前完成，盤中執行
└── Alpha 來源:
    ├── PEAD (60-90 天) - 最持久
    ├── 內線交易 (數天-週)
    ├── 國會交易 (數天-週)
    ├── 負面新聞漂移 (5 天)
    └── 分析師修正 (數天)
```

### L6: 長線持有 (Position)

```
持倉時間: 數週 - 數月
決策頻率: 每週 - 每月
典型策略: 價值投資、成長投資、宏觀配置

數據需求:
├── 延遲: 天級 (週報/月報足夠)
├── 類型:
│   ├── 核心: 財務報表、基本面指標、產業分析
│   ├── 高價值: 13F 機構持股、ESG 評分
│   └── 增強: 宏觀經濟數據
├── 成本: $0-30/月
│   ├── 免費: SEC 10-K/Q/13F、基本面指標
│   └── Financial Datasets $30: Segmented Revenue
└── 基礎設施: 最低要求

AI Agent 適用性: ✅ 適用
├── 決策窗口: 數週，可進行最深度分析
├── 預抓比例: ~95%+
├── LLM 角色: 深度研究報告生成

```

## 推薦策略

### 針對你的情況 (每數小時-每天)

你的交易頻率屬於 **L4 (短波段) 到 L5 (中波段)** 之間，這是 AI Agent 的**最佳適用區間**。

#### 推薦: L4-L5 混合策略

```
策略組合:
├── L4 短波段 (40%): 每數小時決策
│   ├── 訊號: Options Flow 異常、即時新聞反應
│   ├── 持倉: 數小時-數天
│   └── 數據更新: 每 1-4 小時
│
└── L5 中波段 (60%): 每日決策
    ├── 訊號: 內線交易、國會交易、PEAD、分析師修正
    ├── 持倉: 數天-數週
    └── 數據更新: 每日批次

為何這個組合最佳:
├── 充分利用 AI Agent 深度分析能力
├── 數據成本可控 (~$80-150/月)
├── 預抓比例高 (~70-80%)
├── Alpha 訊號覆蓋廣 (從數小時到數月)
└── 不與 HFT/專業日內交易者直接競爭
```

## 總結

### 關於「短線」定義的澄清

```
傳統定義:
├── 短線: < 數天 (包含 L2-L4)
├── 中線: 數天-數週 (L5)
└── 長線: 數週+ (L6)

數據策略定義 (更實用):
├── 即時依賴型: L1-L3 (需要即時數據流)
├── 批次友好型: L4-L5 (可預抓大部分數據) ← 你的位置
└── 完全預抓型: L6 (幾乎100%可預抓)
```

### 你的最佳選擇

| 問題 | 答案 |
|------|------|
| 我的頻率 (每數小時-每天) 屬於? | **L4-L5 (短波段-中波段)** |
| 這算短線嗎? | 傳統定義是，但**不是**即時依賴型短線 |
| 需要即時數據嗎? | **不需要**，15分鐘延遲或每小時批次足夠 |
| AI Agent 適用嗎? | **最適用**區間 |
| 可預抓比例? | **70-90%** |
| 建議月費? | **$80-150** |

### 最終建議

```
推薦策略: L4-L5 混合
├── L4 部分 (40%): 每 1-4 小時更新，捕捉 Options Flow 和新聞反應
├── L5 部分 (60%): 每日批次，捕捉內線交易、國會交易、PEAD
├── 預抓比例: ~75%
├── 月費: ~$80-150
└── AI Agent: 完全主導決策

不建議:
├── 嘗試 L3 或更短 (成本高、AI 優勢小)
├── 追求即時 Options Flow (批次抓取足夠)
└── 訂閱即時新聞 (新聞發布時價格已反應)
```
